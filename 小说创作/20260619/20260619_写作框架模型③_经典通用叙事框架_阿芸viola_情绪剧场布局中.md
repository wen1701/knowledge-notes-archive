## 🎯 核心提炼版

### 三幕剧结构：经典通用叙事框架

- **整体逻辑**：铺垫建立世界 → 冲突升级挣扎 → 决战收尾闭环

#### 第一幕：开端（铺垫）
1. **展示主角**：日常状态、生活环境、性格、欲望与缺陷
2. **构建世界**：交代世界观、人物关系，帮助读者代入
3. **煽动事件**（触发点）：打破平静生活，故事正式启动
4. **结尾**：主角做出选择，决心冒险，衔接第二幕

#### 第二幕：发展（对抗）——篇幅最长
1. **进入新环境**：遭遇连续小挑战，不断尝试突破
2. **冲突升级**：敌人变强、压力加大，主角反复受挫
3. **中点转折**：经历重大失败/背叛，跌入人生低谷
4. **反思成长**：寻找破局方法，为高潮蓄力

#### 第三幕：结局（解决）
1. **高潮决战**：直面最大危机，完成最终对抗
2. **核心矛盾爆发**：做出关键抉择
3. **冲突化解**：危机解除
4. **尾声**：主角蜕变，回归生活，交代人物归宿

## 💡 关键要点总结

- **三幕结构**：开端(铺垫) → 发展(对抗) → 结局(解决)，是影视和文学中最通用的叙事模型之一。
- **核心动力**：通过“煽动事件”打破平衡，不断升级冲突，逼迫主角成长，最终完成蜕变。
- **关键转折**：第二幕的“中点转折”是故事的情绪低谷，也是主角觉醒和高潮蓄力的关键节点。

## 🛠️ 实践建议

1. **对照检查**：写完后，用三幕结构框架检查你的故事脉络，看看是否有缺失的环节。
2. **重点打磨**：重点关注“煽动事件”的吸引力（第一幕结尾）和“中点转折”的情绪冲击力（第二幕中间）。
3. **节奏控制**：第二幕篇幅最长，要确保冲突是阶梯式上升的，可以设计3-5个小挑战让主角一次次尝试和失败。
4. **读者代入**：第一幕要尽快展示主角的欲望和缺陷，让读者产生共情和代入感。
