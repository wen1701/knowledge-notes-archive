## 🎯 核心提炼版

### 一、短篇小说过稿通用公式框架（框架版）

- **导语（50–150字）**：必须包含悬念/反转/爽点，写出爆点。  
  **公式**：受辱 + 反击 + 打脸

- **正文（按字数分阶段）**：
  1. **0–2000字：拉情绪**  
     主角受辱，被欺负/嘲讽/贬低。
  2. **2000–5000字：爆发情绪**  
     主角打脸，拉小高潮铺垫反击，把最气人的片段放上。
  3. **5000–8000字：反转**  
     多来反转，如反派各种陷害/阻挠，主角见招拆招，打脸反派爽点密集。
  4. **8000–10000字：让读者解气**  
     反派作死不能，开始低声下气装可怜，主角可假意和好/信任。
  5. **10000–12000字：大结局**  
     再次反转，主角大仇小仇一起报，怎么爽怎么来，读者看得开心。

### 二、不同题材的剧情公式（公式版）

- **虐文公式**：前期美满 + 中期冲突升级 + 后期情感高潮（死亡/分离/重归于好等）
- **甜文公式**：甜蜜初遇 + 日常相处 + 轻微障碍 + 完美结局
- **成长文公式**：主角平凡起点 + 面临挑战 + 成长与转变 + 实现目标
- **悬疑文公式**：神秘事件 + 侦探/主角介入 + 线索搜集 + 真相大白
- **科幻小说公式**：未来设定/外星世界/时空穿越 + 科技元素/超能力 + 冲突与挑战 + 人性探索/社会反思 + 解决方案/未来愿景
- **穿越文公式**：现代/异界主角 + 穿越契机（意外/主动） + 古代/异界背景 + 文化/能力冲突 + 成长与适应 + 情感纠葛 + 历史/命运改变 + 回归/留驻选择

## 💡 关键要点总结

- 导语必须用“受辱+反击+打脸”的公式制造爆点。
- 正文按0–12000字分为5个阶段：拉情绪→爆发情绪→反转→解气→大结局。
- 不同题材（虐文、甜文、成长文、悬疑文、科幻文、穿越文）各有固定公式，可直接套用。

## 🛠️ 实践建议

1. **先选定题材**：根据你想写的类型（虐/甜/悬疑等）直接套用对应公式大纲。
2. **分段填充**：按12,000字规划，先写导语（50–150字抓住读者），再按阶段一步步推进剧情。
3. **注重反转密度**：在5000–8000字阶段多设置反转，保持爽点密集。
4. **结尾要爽**：大结局要让人解气，不能心软放过反派，读者越爽越好。
5. **检查公式完整性**：每个阶段都检查是否包含了“受辱→反击→打脸”的闭环，以及题材公式中的关键元素是否齐全。
