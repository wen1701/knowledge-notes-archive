## 🎯 核心提炼版

### 1. 核心发现：同一作者的两篇爆款甜文，题材不同但内核完全一致
- **现言甜文**：青春治愈、安静甜（暗恋智宇文）
- **古言甜文**：穿越沙雕、热闹疯（傲娇搞笑文）
- **共同点**：都是女频甜文，均使用同一套创作公式，实现无缝切换

### 2. 爆款公式拆解（四层内核相同）

#### ① 女主内核：沙雕吐槽型
- 脑洞大、射死（社死）害羞
- 内心细腻但外表疯癫
- 穿越/吐槽/脑补/敢跟皇上抬杠
- 外表怂，内心疯，自带吐槽弹幕

#### ② 男主内核：外冷内热型
- 口嫌体正直，表面高冷霸道
- 内心温柔沙雕
- 表面帝王傲娇，内心占有欲强
- 嘴硬虫（嘴硬宠），暗中关注，只对女主特殊

#### ③ 感情模式：双线暗恋 + 女主脑补误会
- 现言：女主以为单恋，其实男主早就喜欢她
- 古言：女主以为自己是把子（把兄弟），皇上有白月光，结果皇上早就盯上她
- 核心套路：女主脑补 + 男主默默宠 + 反差甜 + 沙雕搞笑

#### ④ 喜剧结构：脑洞 + 射死（社死） + 翻车
- 现言笑点：作文散出、头掉西湖、还抵抖杀鱼（？）
- 古言笑点：吃笔看不懂、算数勤谈的像拘幕头、千马被马六、脑补皇上白月光
- 笑点逻辑：一本正经胡说八道 + 大型社死现场

### 3. 职业写手的秘密：换皮不换核
- 晚川拥有一套万能女频甜文公式：沙雕女主 + 高冷傲娇男主 + 女主疯狂脑补误会 + 男主暗中宠 + 无恶毒女配 + 无虐轻宠 + 全程搞笑甜 + 第一人称内心戏拉满
- 将这套公式套进现代校园 → 现言甜文
- 将这套公式套进古言穿越 → 古言甜文
- 题材换皮，内核不变，稳定出爆款

## 💡 关键要点总结

- 同一套公式可套用任何时代、任何题材（古言/现言）
- 人设稳定：沙雕女主 + 外冷内热男主
- 情感模式固定：女主脑补误会 + 男主暗中宠
- 喜剧结构统一：脑洞 + 社死 + 翻车
- 无恶毒女配、无虐，全程甜笑

## 🛠️ 实践建议

1. **提炼自己的万能公式**：先写一篇现代校园甜文，再尝试将相同人设、情感模式、笑点结构移植到古言背景。
2. **保持人设稳定**：女主性格（沙雕吐槽）、男主性格（外冷内热）不能变，只换场景和时代元素。
3. **笑点逻辑一致**：无论古代现代，都靠“一本正经胡说八道”和“大型社死现场”制造笑点。
4. **第一人称内心戏**：强化女主脑补和吐槽，拉满读者代入感。
5. **避免虐和恶毒配角**：保持轻松甜宠基调，专注男女主互动。

## 📜 完整内容精排版

在插一篇晚穿的沙雕古言填文，插它是因為有4.1萬點讚，2.5萬收藏，還是女評文，但與前一篇不太一樣。前一篇是現言，這一篇是古言。前一篇填的安靜，是青春暗鏈智宇文；這一篇填的熱鬧，是古言穿越澳交沙雕文。一個作者為什麼能寫出不同題材的故事？因為它是靠創作公式示範，它能無縫切換，因為兩篇內合完全一樣。我們這一節就來講講同一個作者的兩套爆款生產模式。

1. 女主內合完全一樣，都是沙雕吐槽型。杜建朝腦洞大，射死害羞，但內心細足。難之異穿越吐槽腦補，敢跟黃泉抬抗，都是外表宋內心風，自帶吐槽淡目。

2. 男主內合完全一樣，外冷內熱，口閒體政治看壞。表面高冷血霸，內心溫柔沙雕。將繼表面帝王澳交，內心戰友欲強，都是罪硬蟲暗中關注，只對女主特殊。

3. 感情模式完全一樣，雙性暗鏈加女主腦補誤會。前一篇女主以為自己單戀，男主早就喜歡他。這一篇女主以為自己是把子，皇上有白月光，結果皇上早就定上他，蟲個不行。核心套路統一：女主腦補，男主默默蟲，反差田殺雕搞笑。

4. 喜劇結構完全一樣，靠腦洞加射死加翻車制造效點。前一篇效點穿作文散出頭掉西湖還抵抖殺魚；這一篇效點吃筆看不懂算數勤談的像拘幕頭千馬被馬六腦補皇上白月光。效點邏輯一模一樣：一本正經的胡說八道，加大型射死現場。風格變，底層攻勢不變。
