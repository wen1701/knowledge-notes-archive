## 🎯 核心提炼版

### 核心观点
- 人设不能靠“想”，必须让人物从剧情里“生长”出来。
- 作者自创两个公式，分别针对**主角/重要配角**和**普通配角**。

### 公式一：主角和重要配角 → 服务剧情 + 多个标签
1. **先确定故事类型**：根据你想写的故事（如甜狗虐恋、家族复仇等）确定主角的核心性格。
   - 甜狗虐恋 → 主角应敢爱敢恨、深情、傻、无感情束缚。
   - 家族复仇 → 主角应聪明勇敢、隐忍互短、以家族为核心。
2. **让主角从故事里长出来**：主角要带领故事走，不能乱做设定。
3. **添加小标签**：在核心性格基础上，加2-3个标签让人物更立体（如内心敏感、知恩图报、努力上进）。
4. **为标签赋予“可寻因”**：每个标签都要有背景故事，形成人物小传。
   - 例：甜狗主角脆弱敏感 ← 从小父母离婚；知恩图报 ← 被女生帮助才爱上女主；努力上进 ← 从小家庭不好只能靠自己。
5. **最终形成人物小传**：一个从小家庭贫困、父母离异、缺爱脆弱的主角，被女生帮助后爱上女生，但发现女生不爱自己，决定放下感情努力奋斗。

### 公式二：普通配角 → 服务主角 + 单一标签
- 所有配角都用来帮助主角完成剧情，所以配角服务于主角。
- **主角缺什么人，就设置什么人**。
- 每个配角只需一个标签（如反派、好兄弟、校花），作用单一，无需详细刻画。
- 例：甜狗主角需要刺激 → 设置反派追求女生成功；需要点醒 → 设置好兄弟开导；需要逆袭机会 → 设置校花爱上主角制造情敌。
- 配角用“一个表现就够了”：反派负责消账保护，兄弟负责有请，校花负责一笑花下两天。读者视角始终聚焦主角。

## 💡 关键要点总结

- 人设从剧情生长，主角服务剧情，配角服务主角。
- 主角公式：确定故事类型 → 核心性格 → 加多个标签 → 赋予背景故事 → 形成人物小传。
- 配角公式：主角缺什么 → 设置单一标签配角 → 只用一个表现完成功能。

## 🛠️ 实践建议

1. **写主角前先明确故事类型**：你写的是甜宠、复仇、升级流还是其他？主角性格必须匹配故事基调。
2. **给主角加标签时，每个标签都要有“为什么”**：比如“敏感”是因为童年创伤，“努力”是因为家境贫寒。这样人物才有根。
3. **配角设计时，先列主角成长需要的“功能”**：需要敌人？需要导师？需要朋友？然后每个功能配一个单一标签的配角，不要给配角写复杂背景。
4. **检查你的角色**：如果主角的标签没有剧情支撑，或者配角抢了主角风头，立刻调整。

## 📜 完整内容精排版

心想说千万不要去向人设，不用抱款，十本改编短距的经验告诉你真相，股市民想是写不好人设的。必须要让人物从你的剧情里面生长出来。如果你们写的人物生意总觉得不够信任，那就看看我自创的两个公寺，内容非意长干，建议点赞收藏。

第一个公寺是用于主角和重要配决的：服务剧情加多从标签。咱们写主角要先确定好你要写什么样的故事。如果你想写一个甜狗决情的故事，主角就应该感爱感恨、亲情傻和无悲感情束缚。如果你想写一个家族在先解决奇的故事，主角应该聪明勇敢、引人互短、以家族为核心。就是要让主角从故事里长出来，让他去带领故事走，不能复乱做设定。

但是人都是复杂多样的，所以咱们服务好去行之后，应该再给主角添加一些小标签，让他更多面化，更像一个活生生的人。比如一个甜狗，他不仅是感爱感恨，他也可以是内心敏感，也可以是知恩图报，也可以是努力上进。那么后面三个就是咱们给他提了标签。我们提完标签后，最好让标签有即可寻，让人物再形成一番故事，也就是我们说的人物小赚。依旧拿甜狗举例：他脆弱敏感，可不可以是从小父母离婚受到伤害呢？他知恩图报，可不可以是被女生帮助所以才爱上女主呢？他努力上进，可不可以是因为从小家庭不好，明白只能靠自己努力呢？那么现在我们就有了一个人物小赚：一个从小家庭贫困、父母离意、缺爱脆弱的主角，被女生帮助后爱上女生，但亲行之后发现女生根本不爱自己，所以决定放下感情努力奋斗。

OK呀，现在你已经学会如何设计主角了。咱们接下来第二个公司是对于不那么重要的配绝。配绝的公司很简单：就是服务主角加单一标签。除了个别群项目以外，所有的配绝都是用来帮助主角去完成剧情的。所以主角服务于剧情，那么配绝就应该服务于主角。主角缺什么人，就要来什么人。话不多说，咱们继续去努力。我为了让大家通俗一懂，觉得你只比较熟悉白白建议一就是田狗主角。你想让主角从田狗到决心，是不是需要人来刺激一下？咱们直接设置一个反派的配绝，锋乱追求女生成功。再来主角需不需要人来点醒他？直接设置一个好兄弟一直在劝主角开导主角。继续啊，主角努力上几后，需不需要给他讲运呢？直接来一个比女生还漂亮学习还好的笑话，或者是亲媒爱上主角，花香敌质来吗？也很符合逻辑。那么三个配绝是不是也就自然地生长说了？还要主要配绝就用一个表现就够了：附而来消账保护兄弟，有请有一笑花下两天。他们的作用仅仅是帮助主角，没有必要协认详细，要让独者的视角一直聚焦在主角身上。
