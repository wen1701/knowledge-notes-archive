## 🎯 核心提炼版

### 一、核心概念：3秒留人
- **定义**：小说开篇3秒内，用户没有明显排斥且觉得有点意思，会持续阅读。
- **数据**：3秒留人成功，到10万字时留存率约30%，作品必爆。
- **例外**：话不能说满，但大概率成立。

### 二、如何实现3秒留人

#### 1. 开篇直接抛冲突（而非背景介绍）
- **错误做法**：交代世界观、人物身份、矛盾、危机、修炼等级等（像产品说明书）。
- **正确做法**：直接抛出人物冲突、金手指、反差。
- **案例**：杀夫之仇+多妻战+刘老三：“我刘如烟你配不上？” + 金手指：穿越+苦修一天等于练功一年。

#### 2. 冲突的三种类型
- **画面感**：人与人之间的冲突（如杀夫之仇、多妻战）。
- **烈气感**：不该冲突的地方冲突了（如分手时女主说“你根本不懂爱”，反派说“把爱分成八份”）。
- **环境描写**：人与物的冲突（如环境描写），但开篇500字内不需要，因为用户不需要知道环境。

#### 3. 三个拒绝
- **拒绝慢热**：谈恋爱都不能慢热，慢热用户会划走。
- **拒绝邪警**（疑似“拒绝邪警”应为“拒绝邪警”？原文如此）：拒绝传统的、常见的、普通的警（可能指套路）。
- **拒绝做自我介绍**：不要讲家族背景、世界排名等无意义信息。

#### 4. 核心比喻：石头砸水花
- 开篇500字就像把石头搭在水面上，碰出水花——这个画面才是留人的关键。
- 流程：吃粉（？原文“吃粉”可能为“吃透”或“吃粉”？） → 讲这块儿 → 做个拉钦（可能为“拉亲”或“拉琴”？） → 做个剑身（可能为“剑身”或“健身”？） → 把石头放在肩膀上推出去 → 石头搭在水里碰水花。

## 💡 关键要点总结

- 开篇3秒必须用冲突抓住用户，不要铺垫背景。
- 冲突要有画面感（人与人）和烈气感（不该冲突的地方冲突）。
- 拒绝慢热、拒绝套路、拒绝自我介绍。
- 开篇500字就是石头砸水花的瞬间，直接引爆。

## 🛠️ 实践建议

1. **写开篇时，先写冲突场景**：比如分手、杀父、金手指觉醒，不要写“这个世界是怎样的”。
2. **检查前500字**：是否出现了“我是谁”“我家在哪”“修炼等级”等无效信息？删掉。
3. **制造烈气感**：让角色在“不该冲突”的场合爆发冲突（如分手时翻旧账、反派当面羞辱）。
4. **用“石头砸水花”自检**：你的开篇是否像石头入水一样瞬间激起水花？如果不是，重写。
5. **领取大纲**：加入粉丝群获取详细大纲，参考新书《二集我小子陷阱》学习。

首先第一点就是开篇有直接抛什么？开篇也直接抛出一个观点来的。这观点是什么？是人物冲突，是金手指，是反差。不要小破电，就是很多人不明白这句话。我再讲一句就明白了。又多少人写的第一本小说里面开篇交代的事：这个世界是一个怎样的事件，这个人是一个怎样的人，这个矛盾是一个怎样的矛盾，这个危机是一个怎样的危机。我问一下，我需要知道吗？我需要知道这个角色的身份、心意已经所有问题吗？我不许。为什么？这些内容比减完了大概就是一千字。他特别像一部作品的时候使用说明书，而且这作家很贴心，把修炼等级也给我放在五百字里面了。这个修炼等级一共分为九层九重天，最低层的是练气，最高层的是至高，至高上面是无上，无上是班是胜人。我几头白点看了就为了了解一下这个世界的修炼观点，我有意义吗？没有意义。所以我们应该做这么做。冲突最后这么做突。我们见到说你再不会你就做最简单的什么？最简单杀腐之初多期战。刘老三，我刘如烟你配不上？好，分九分。为什么？我穿越过来的，刚刚我来执行了一个金融质一日等于一年，也就是这种苦休一天等于我练习功法练了一年。哎，这就是冲突。接着就是苦休，比如说俱博先盘能够提高砸物的台质相亲变强，我相亲一次能够给十万到五百万。翻转，翻转是什么？开票做翻转。刘如烟不识好歹，刘如烟走了。等一项怎么了？这有些年我刘家给你刘家给了不少东西吧。那都是你强迫我给我的。那我现在不强迫了，把东西给我还回来。看翻转也出现了，对不对？我做不强了吗？刘如烟是刘自家族，刘自家族在中州排名老二。这个事件一共有六百七十二个州，中州是到处之一。中我做这种不强了吗？没有吧。这种不强有意吗？是不是没有意义？而接下来黄妍我版子必须有画面感、练结感、注意力必须有什么叫画面感？简单杀负之初多期之恨人与人的冲突叫画面感。那听你的意思还有人与物与物的冲突？有环境描写就是人与物，物就是那个环境。今天突门天使南的草之星的雪石綠的突门，我溜大点玩水泥有小鱼在这土泡泡，我在哪儿都能看到这个环境，我就不需要你来讲。所以五百子里面的画面感本身就是人与人之一的冲突。而这个烈气感是什么？是不该冲突的它冲突了。比如这种要是刘如烟，我俩分手吧，你根本不懂爱，你根本不能遵征我，你每个月你知道给我扎千一千万。再比如我们做个反派炸难的这个炸难说了怎么了我虽然该你那五六七八个闺蜜悠然，但为啥爱你怎样分成八份不就行了？你个见人就是狡情女主怕大嘴巴的冲过去别忘了你的身份是怎么来的。我六家可给他你用千万创业资金培了你那兄弟姐妹也培了，你还敢有点给我在这儿倒闭倒闭倒闭也行都可以。那里把南天女婷里面多多烈气，烈气事不该冲突的地方冲突了。记住这里自己回味一下。
