## 🎯 核心提炼版

### 分层写作五步法（附案例）

**案例原始一句话**：他站在吊桥上。

**第一步：用一句话构思一个场景**
- 一句话：他站在吊桥上。

**第二步：扩展这句话，给读者更多信息**
- 扩展后：他站在摇摇欲坠的吊桥上，手心冒汗。

**第三步：加入背景信息和心理活动**
- 加入背景：这里曾是他儿时的秘密花园。
- 加入心理：现在只剩下风吹过的声音和他心中不安的回忆。
- 完整句：他站在摇摇欲坠的吊桥上，手心冒汗。这里曾是他儿时的秘密花园，现在只剩下风吹过的声音和他心中不安的回忆。

**第四步：加入场景描写，增加氛围感**
- 场景描写：吊桥下是深不见底的峡谷，一阵风吹过。
- 完整句：吊桥下是深不见底的峡谷，一阵风吹过。

**第五步：场景描写结束，焦点迅速回到人物身上，带入下一个场景**
- 焦点回人物：他深吸一口气，平定心跳。
- 带入下一个场景：就在这时，远处传来一阵熟悉的歌声，他抬头望去，只见对岸牢笼上，一个身影正轻轻摇晃，正是他儿时神秘失踪的好朋友。

### 方法论总结
- **分层逻辑**：从一句话 → 扩展细节 → 背景心理 → 环境氛围 → 人物行动与情节推进。
- **核心原则**：每一层只增加一个维度的信息，避免一次性堆砌；场景描写后必须迅速回到人物，保持叙事节奏。

## 💡 关键要点总结

- 分层写作：一句话构思 → 扩展信息 → 背景心理 → 场景氛围 → 焦点回人物。
- 案例示范：从“他站在吊桥上”逐步丰富到包含环境、心理、情节转折的完整场景。
- 技巧：场景描写为人物服务，描写后立即回到人物动作或心理，推动剧情。

## 🛠️ 实践建议

1. **练习步骤**：找任意一句话（如“她坐在窗前”），按五步法逐层扩展，每层写一句，不贪多。
2. **注意平衡**：背景和心理活动不要过长，控制在1-2句；场景描写要简洁，突出氛围感。
3. **焦点切换**：场景描写结束后，立刻用人物动作（如“深吸一口气”“抬头望去”）拉回读者注意力，并自然过渡到下一场景。
4. **反复修改**：写完五层后，通读检查是否流畅，可适当调整顺序或删减冗余。
