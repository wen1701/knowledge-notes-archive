## 🎯 核心提炼版

### 一、构图与镜头
- **视角**: 正上方垂直俯拍（上帝视角）——剥夺个体主体性的终极视觉形式。
- **景别**: 广角远景——放大人类在宏大系统面前的渺小与虚无。
- **人物布局**: 完全摊开的面（躺卧）——人类在命运前彻底放弃抵抗的象征。
- **环境占比**: 绝大部分空间留给环境，前景+背景倒窝的人物。

### 二、光影与色调
- **光源**: 明天慢摄的自然光（天然柔光）——绝对主导。
- **辅助指令**: 去除人工辅助光与数字锐化。
- **色调**: 极高压抑感的高色温冷清灰色，加湿润的混凝土色、脏白胶质色，并赋予它暗部生态、灰色胶片质感。

### 三、生成流程
1. 脑海中浮现画面 → 2. 拆解视觉心理学要素 → 3. 撰写精准提示词 → 4. 将提示词甩给AI → 5. 获得直击灵魂的画面 → 6. 提炼为可复用的方法体系

### 四、底层逻辑（视觉心理学）
| 元素 | 心理效应 |
|------|----------|
| 上帝式俯拍 | 剥夺个体主体性 |
| 广角远景 | 放大渺小与虚无 |
| 人物摊开/躺卧 | 彻底放弃抵抗 |
| 单色（极低饱和度） | 吞噬一切喧嚣的视觉真空 |
| 青灰冷色调 | 生命力流失的视觉温度 |

## 💡 关键要点总结

- 用上帝视角俯拍+广角远景+躺卧人物营造孤独宿命感。
- 自然光为主，去除人工辅助光和锐化，保持低饱和度冷灰色调。
- 建立自己的视觉心理学底层逻辑库，把单次灵感转化为可复用的方法体系。

## 🛠️ 实践建议

1. **学习该提示词结构**：镜头角度→人物布局→环境比例→光影设定→色调描述。
2. **建立个人视觉词典**：将每次成功出图的元素与心理效应对应整理。
3. **实验参数**：尝试不同色温、饱和度、胶片质感组合，记录效果。
4. **反复迭代**：生成后再分析画面，优化提示词中的心理暗示词。
