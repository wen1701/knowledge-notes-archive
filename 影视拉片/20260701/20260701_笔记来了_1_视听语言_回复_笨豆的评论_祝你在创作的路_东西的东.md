## 🎯 核心提炼版

### **核心主题：视听语言系统学习笔记**
这份笔记是一个完整的《视听语言》课程知识体系，涵盖了从影像基础、景别角度、焦距景深、镜头运动、轴线原则、布光技巧（光位与色温）、色彩理论、声音剪辑到蒙太奇理论的方方面面。

### **关键概念与知识点**

- **影像与感知基础**：电影元素（影像、声音、剪辑）；视觉暂留与完形心理学（格式塔，大脑的“自动补足”能力）。
- **构图与景别**：
    - **构图三原则**：美学原则、主题服务原则、变化原则。
    - **构图微原则**：主体不居中、水平线不居中、主体不孤单、主陪体分明。
    - **景别序列**：大远景 → 远景 → 大全景 → 全景 → 中景 → 中近景 → 近景 → 特写 → 大特写。
    - **景别划分标准**：远景（广阔场面）、全景（人物全身）、中景（膝盖以上）、近景（胸部以上）、特写（颈脖以上或细节）。
- **焦距与景深**：
    - **景深三要素**：**光圈**（反比，光圈值越大，景深越短）、**焦距**（反比，焦距越长，景深越短）、**物距**（正比，物距越长，景深越长）。
    - **镜头分类**：定焦（中焦35-50mm、长焦50-250mm、短焦/广角<35mm）与变焦。
    - **景深镜头（深焦距）**：代表《公民凯恩》的全景深镜头。
- **镜头运动**：
    - **运动类型**：推、拉、摇、移、跟、升、降。
    - **推镜头**：突出主体、强化情绪（特写→激动，全景→冷静）、模拟视线。
    - **拉镜头**：拉出环境、建立联系、换场收尾。
    - **摇镜头**：固定支点做扇形或圆形运动（角度一般>90°，<180°）。
    - **移镜头**：运动工具上移动拍摄（前跟、后跟、侧跟）。
    - **升/降镜头**：本质类似移镜头的“侧跟”，但方向变为竖直。
    - **长镜头**：安德烈·巴赞提出，理论挑战蒙太奇，强调时空连贯性与真实。分类有记录式、叙事性、时间长镜头。鼻祖：《北方的纳努克》。
- **轴线与机位**：
    - **轴线的定义**：被摄对象的视线方向、运动方向或与他对象人物之间的关系所形成的一条假想直线。
    - **轴线分类**：关系轴线（两人以上的连接线）、运动轴线（运动方向）、方向轴线（视线）。
    - **轴线规律（180度法则）**：拍摄同一场面主体时，机位必须在轴线同一侧，否则“跳轴”。
    - **合理越轴方法**：特写镜头越轴、主观镜头越轴、大景别越轴、插入中性镜头（空镜头）越轴。
    - **九机位法（标准机位法）**：顶角（总角度，最核心）、外反拍（过肩，展现关系）、内反拍（单人，对立感）、垂直（拍动作）、骑轴/平行（主观视线）。
- **布光与色彩**：
    - **三点布光法**：主光 → 辅助光 → 逆光（轮廓光，使主体立体）。
    - **水平光位**：正面光 → 前侧光 → 正侧光(90°) → 侧逆光 → 逆光(180°).
    - **垂直光位**：顶光(90°) → 高位光 → 中高位 → 中位 → 中低位 → 水平(0°) → 低位/脚光。
    - **色温**：单位K（开氏温标）。低色温偏红（如日出日落2000-3000K），高色温偏蓝（如阴天6800-7500K，晴空10000-20000K）。
    - **色彩情绪**：暖色（红橙黄绿）突出、活力、暴力；冷色（淡绿、青、蓝、蓝紫）安静、孤独、忧郁。
- **声音与剪辑**：
    - **声音录制**：前期录音、后期录音（常规）、同期录音（真实感，纪实风格）。
    - **声画关系**：**声画同步**（严格匹配）、**声画分立**（平行发展）、**声画对位**（情绪对比/冲突，如暴力场面配轻快音乐）。
    - **影视声音分类**：**人声**（对白、独白、旁白）、**音乐**（有声源、无声源）、**音响**（自然、人造）。
    - **声音剪辑**：**平行剪辑**（同位法，平稳、庄重）、**交错剪辑**（串位法，活泼、明快）。
- **剪辑理论**：
    - **经典剪辑（无缝剪辑）**：核心是建立稳定的空间感、连续的时间线、清晰的逻辑关系。
    - **连贯性剪辑原则**：时间的连续性、空间的连续性（180度轴线）。
    - **非连贯性剪辑**：打破时空连续性，导演有意创造风格。
    - **格里菲斯的剪辑理论**：闪回（交代过去）、平行剪辑与“最后一分钟营救”、加速剪辑。被尊为“美国电影之父”。
    - **蒙太奇**：
        - **表现（表意）蒙太奇**：抒情（《乡村女教师》）、隐喻（《战舰波将金》）、象征（《甜蜜的生活》）、对比（《小鞋子》）。
        - **叙事（叙述）蒙太奇**：平行（《薇罗尼卡的双重生活》）、交叉（《一个国家的诞生》）、错觉（《51号兵站》）、积累（《战舰波将金》）、重复（《魂断蓝桥》）。
    - **动接动静接静**：静接静（纪录片头尾）、动接动（动作连贯，如追捕）、动接静（动作做完后切静）、静接动（后一镜头动作即将开始时切）。
- **视点**：客观（中立、全知）、主观（画面变形，介入场景）、导演（引导观看）、间接主观（卷动情节）。

## 💡 关键要点总结

1.  **轴线是空间秩序的“红绿灯”**：拍摄对话或运动场景时，机位必须保持在180度轴线一侧，否则观众会迷路。越轴请用特写、主观镜头或空镜头。
2.  **景深三圈定“焦点”**：光圈大景深短（虚化），焦距长景深短（特写），物距短景深短（近处物体对焦）。《公民凯恩》的全景深镜头是摄影经典。
3.  **三点布光是“立体的灵魂”**：主光定基调，辅光补阴影，逆光（轮廓光）把主体从背景中“拉”出来。
4.  **色温是画面的“冷暖情绪”**：低色温（偏红）暖，常用于温馨；高色温（偏蓝）冷，用于忧郁、科技或恐怖。
5.  **“动接动”与“静接静”是流畅剪辑的“铁律”**：运动镜头之间衔接要保持方向、速度一致；静止镜头之间衔接要统一景别或关系，避免“跳”。
6.  **蒙太奇是电影的“语法”**：叙事蒙太奇讲“故事怎么讲”，表现蒙太奇讲“观众怎么感受”。

## 🛠️ 实践建议

1.  **拉片练习**：找一部经典电影（如《罗拉快跑》、《红高粱》或《公民凯恩》的片段），专门分析它的**轴线使用**、**景别变化**和**三点布光**，并在纸上画出机位图。
2.  **日常拍摄**：下次用手机/相机拍人时，尝试运用“动接动静接静”律进行剪辑，观察效果。在室内环境，尝试移动灯具（台灯/手电筒）实现简单的三点布光，感受光对人物立体感的影响。
3.  **学会用景别说话**：情绪激动用推镜头和特写；表现孤独或环境用拉镜头和远景；悬疑感用摇镜头和长焦。
4.  **注意声音**：下次剪辑Vlog时，尝试使用“声画对位”，比如在平静的画面下放紧张的音乐，或在吵闹的画面下放慢速音乐，打破常规创造新情绪。
5.  **避免“跳轴”**：在拍摄对话时，始终记住180度线，确保两个对话者始终在画面的左右两侧位置一致，否则观众会以为他们在看不同方向。

5. **构图**
①概念：即画面的结构和布局，是各形象元素的联结关系，画框的存在是构图的基础。
②注意事项：(1) 线条 (2) 几何形状：增加景深，制造纵深和空间感 (3) 水平与垂直关系 (4) 位置、中心、平衡：有效的适用画幅空间，画面人物位置，放置均匀 (5) 各元素的长短、大小、面积的对比 (6) 光影、颜色的变化
③构图时要遵循的原则
(1) 美学原则
1>主体不要居中
2>水平线不要上下居中，避免一分为二的切割画面
3>色调、布光不要一分为二切割画面
4>主体不要孤单
5>主体、陪体应主、陪分明，不应喧宾夺主
6>人或物的断x线不应一字排开（注：“x”处字迹模糊，疑似“裂”或“点”）
7>
8>水平线及景物连接线不应歪曲不稳
9>人不要完全正面出现，与镜头形成一定角度
(2) 主题服务原则
1>为了表现好主题，努力设计最合适、最舒服、最具视觉美感的构图，如对比、均衡、对称、集中等
2>要刻意破坏画面构图的美感，即所谓的不规则构图
3>如果某个优美的画面，与整个影片的风格，主题不符甚至妨碍了影片主题的表达，必须忍痛割爱。
(3) 变化原则：对于由于千万个单个画格组成的整部影片的构图而言，所要遵循的原则是变化原则。注意：景别、角度、运动、照明、色彩都属于构图方式变化
第三节 景别与角度
1. 机位：也叫镜位，决定拍摄景别与角度（景别和角度是构成影像的基础）
2. 机位的选择和设置：建立镜头，主镜头、切入镜头、双人镜头、反拍镜头、过肩镜头、反应镜头
3. 景别：指镜头由于被拍摄物体的距离或焦距不同，所摄取的不同场景的画面。
4. 景别的完整序列：大远景——远景——[黑色粗线遮挡]——全景（人物全景）——中景——中近景——近景——特写——大特写
